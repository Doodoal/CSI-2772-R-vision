Résumé et Quiz - Flots en C++

Résumé

Les flots en C++ sont utilisés pour gérer les entrées et sorties de données. Les principales classes incluent :

  • ostream: Permet d'écrire des données avec l'opérateur <<.
  • istream: Permet de lire des données avec l'opérateur >>.
  • fstream: Permet la lecture/écriture dans des fichiers.

Exemple :

#include <iostream>
using namespace std;

int main() {
    int n;
    cout << "Entrez un nombre: ";
    cin >> n;
    cout << "Vous avez entré: " << n << endl;
    return 0;
}

Les flots peuvent également être connectés à des fichiers via les classes ifstream et ofstream.
